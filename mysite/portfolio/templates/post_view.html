About this project






I'm super happy to bring an update to my latest keyboard.

To make a mechanical keyboard you need to:

  1. Get a metal or acrylic plate cut to sit your keys
  2. We use steel we get from Transfer Flow
  3. Once mounted you need to glue in your switches.
  4. Then solder all the diodes to the switches
  5. Then solder all the wires to all the rows
  6. Then you need to go from all the ends of each row and column to the teensy micro-controller.
  7. After that all you need to do is program it with the teensy loader (https://www.pjrc.com/teensy/loader.html)


This is a rough outline of what needs to be done to make a keyboard. Overall, thanks to the hard work of Dane Eckert and everyone at the Chico State Mechanical Keyboard club I was able to get this keyboard done in a single day.
